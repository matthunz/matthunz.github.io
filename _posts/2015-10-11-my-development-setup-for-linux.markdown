#My Development Setup For Linux

I'm sure you've seen lists like these a thousand times before but hear me out, you may find this list useful. After finally ditching Windows in favor of Arch Linux, here are some of the tools I've found:

 [Atom](https://atom.io/) is by far my favorite editor because of the extensions, themes, and the fact that it's open source. My favorites are Minimap and Snippets.

 [Watch HTTP Server](https://www.npmjs.com/package/watch-http-server) for live reloading while editing.

 [Jekyll](http://jekyllrb.com/) for locally testing before comitting to Github pages.

 [Yaourt](https://wiki.archlinux.org/index.php/Yaourt) for downloading packages from the AUR.

 [BSPWM](https://github.com/baskerville/bspwm) is a great window manager and really improves productivity with shortcuts. However, I wouldn't recommend this if you're just getting started with Linux.

 [GNOME](https://gnome.org/) is a great desktop environment that's very minimalistic and allows for a variety of themes and extensions. The [Arc](https://github.com/horst3180/Arc-theme) theme is by far my favorite.
